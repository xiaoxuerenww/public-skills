# Sprint Doc Template

Use this format when creating `journal/sprint_YYYY_MM_<topic>.md`:

```markdown
# Sprint: <Sprint Name>

**Period**: YYYY-MM-DD to YYYY-MM-DD
**Status**: Active

## Goals
1. Goal A
2. Goal B

## Plan

| Date (Day) | Block 1 (AM) | Block 2 (PM) | Block 3 (Eve) |
|------------|--------------|--------------|---------------|
| MM-DD (Mon) | Task 1 | Task 2 | — |
| MM-DD (Tue) | Task 3 | Task 4 | — |

## Daily Log

```

Daily log entries use this format:

```markdown
### YYYY-MM-DD
- Completed: X, Y
- In progress: Z
- Blocked: W (reason)
- Adjusted plan: (if any changes to upcoming tasks)
```

Omit any sub-bullet that has no content for that day.
