## Installation

These skills are personal and may contain private information. Use locally only.

```bash
# Clone to your skills directory
cd ~/.codex/skills
git clone https://github.com/xiaoxuerenww/private-skills.git temp
cp -r temp/journal .
cp -r temp/personalized-life-coach .
rm -rf temp
```
