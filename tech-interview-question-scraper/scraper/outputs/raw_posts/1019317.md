第二題：
Given an array, find the value N such that left-shifting the array by N will result in a sorted array. Return N or -1 if no such N.
舉例來說, nums = [3, 4, 5, 1, 2]
N = 0, nums left shift by 0，也就是不變
N = 1, nums left shift by 1, nums = [4, 5, 1, 2, 3]
N = 2, nums left shift by 2, nums = [5, 1, 2, 3, 4]
N = 3, nums left shift by 3, nums = [1, 2, 3, 4, 5]，此時nums是sorted的，所以return 3.
另一個例子, nums = [3, 5, 4, 1, 2]
N = 0不變
N = 1, nums = [5, 4, 1, 2, 3]
N = 2, nums = [4, 1, 2, 3, 5]
N = 3, nums = [1, 2, 3, 5, 4]
N = 4, nums = [2, 3, 5, 4, 1]
以上無論shift幾次都不會得出sorted array，所以return -1
我當時的作法是每次shift後檢查是否sorted，是就return current N，否則繼續。
