You are given a sentence and a list of matching phrases. Your task is to wrap each match in <yellow> and </yellow> tags. If any matches overlap or are adjacent, they should be merged into a single <yellow> block.
Input:
A sentence (string)
A list of matches (strings), each of which may appear in the sentence
Output:
Return the sentence with matched regions wrapped in <yellow> and </yellow> tags
Overlapping or adjacent matches should be merged into one tag block
Example:
python
Copy
Edit
sentence = "The quick brown fox jumps over the lazy dog."
matches = ["quick brown", "brown fox jumps"]
Expected Output:
python
Copy
Edit
"The <yellow>quick brown fox</yellow> jumps over the lazy dog."
Explanation:
"quick brown" spans indices [4, 15)
"brown fox jumps" spans [10, 26)
These overlap, so they are merged into one span: [4, 26)
Additional Test Cases:

Edit
