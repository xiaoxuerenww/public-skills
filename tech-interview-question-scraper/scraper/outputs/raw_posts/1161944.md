## Post

题目：k 阶 Fibonacci tree，左子树是 (k-2)，右子树是 (k-1)。用 preorder 从 root=0 开始给节点编号。给 k 和两个节点 id a,b，求 a 到 b 的路径（输出节点序列/思路即可）。
思路（不建树）：
1）先预处理每阶树的节点数 size[k]
size[0]=1，size[1]=1
size[k] = 1 + size[k-2] + size[k-1]
2）利用 preorder 的连续区间定位节点在哪个子树
假设当前子树是 F(k)，根编号是 r：
左子树根 = r+1，范围是 [r+1, r+size[k-2]]
右子树根 = r+1+size[k-2]，范围是后面那一段
写个函数 getPath(k, r, x)：一路判断 x 落左还是右，递归/迭代下去，拿到 root->x 的路径（复杂度 O(k)）
3）分别拿到 root->a 和 root->b 两条路径，找 LCA（从头一起走，最后一个相同点）
a->b 路径 = a 往上到 LCA + LCA 往下到 b（拼接注意 LCA 不重复）
复杂度：O(k)，空间 O(k)。
注意点：区间边界别写错，size 用 long long（k 大可能溢出就提一嘴 BigInt/截断比较）。
