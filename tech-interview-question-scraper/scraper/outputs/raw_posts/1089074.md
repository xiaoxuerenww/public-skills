Problem 2
You are given an array of strings text, where each string represents a word in a document. Identify all words that both start and end with a vowel. Leave the vowels and the first and last characters unchanged. Reverse the substring between these characters for each of these identified words. For all other words, leave the characters unchanged. Leave all words in their original positions and return the modified array of strings.
Example:
For text = ["apple", "banana", "OranGe"], the output should be solution(text) = ["alppe", "banana", "OGnare"].
Explanation:
The word "apple" starts and ends with a vowel. The middle part "ppl" is reversed to "lpp", resulting in "alppe".
The word "banana" does not start or end with a vowel.
The word "OranGe" starts and ends with a vowel. The middle part "ranG" is reversed to "Gnar", resulting in "OGnare".
For text = ["AE", "CodeSignal"], the output should be solution(text) = ["AE", "CodeSignal"].

Problem 4
You are given two arrays of numbers, firstArray and secondArray. Return the length of the longest common prefix (LCP) between any pair of numbers from different arrays or 0 if no common prefix exists.
A prefix of a number is a number formed by one or more of its digits, starting from its highest-order digit. For example, 1 is a prefix of 123456 and 123 is a prefix of the number 1234. A common prefix of two numbers is a prefix that both numbers share. For instance, the longest common prefix (LCP) of 543519 and 54351 is 54351, and there is no common prefix of 123 and 456.
Example:
For firstArray = [141, 285, 12345, 5444] and secondArray = [2, 255, 244, 2446, 26, 544444], the output should be solution(firstArray, secondArray) = 4.
Explanation:
The best pair is 5444 from the first array and 544444 from the second array with the LCP 5444, where 5444 is of length 4.
For firstArray = [123, 285, 255, 54444] and secondArray = [2, 255, 244, 2446, 26, 544444], the output should be solution(firstArray, secondArray) = 3.
Explanation:
The best pair is 544 from the first array and 544444 from the second array with the LCP 544, where 544 is of length 3.
For firstArray = [187, 99] and secondArray = [1999, 1950], the output should be solution(firstArray, secondArray) = 0.
Explanation:
No pair of numbers from different arrays has a common prefix, hence the answer is 0.
