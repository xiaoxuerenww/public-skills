求加米。
个人建议： 题长，先搂一眼做简单的和能理解的题，有的题会给动画的演示，会帮助理解题目。
1.例： list[895,896,897]
判断是不是给到list的区间里的数字，三位数都不相同。output：不相同的个数。 例子里的答案是3个。
2. fish and baits， 用baits去抓鱼，只能捉到比baits大的， 3号的biats只能捉4，5，6.。号的鱼，
baits只能用3次， 问最多能捉多少鱼。input是list。
3.
（不是很明白题意） 游戏设计， input （bubble[list[list[int]]] operations[list[int]]）: 类似于matrix的list，点bubble会消掉相同颜色，之后剩下的元素就会下落， output： 返回operations后的结果
例：
o x o
x 2 x
0 x 0
点击2， 消掉0, 会变成
x
x x x
空 空 空
0  2  0
x  0  x
点击2， 消掉0，其他元素下掉。
4.input是[list[list]], [1, x] x位置有障碍物obstacle，[2,x,y] 代表在x位置之前放了一个y 长度的block，要求看所有[2, x,y]的操作是成功还是失败。
例： [1,5], [2, 3, 1] return true, [1,5]指在5的位置上放了一个obstacle， [2，3，1]从3 开始往前放1个block， [2,3,2]是可以的， 但是[2, 3, 4] 不行。不确定是从0 开始还是从1 开始。
