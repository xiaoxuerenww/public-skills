## Post

以下内容需要积分高于 220 您已经可以浏览
Given a collection of N graphs, merge them into one graph by adding minimum number of new edges. Output finally generated graph.
Need to ensure complete randomness.
在理解randomness上花了些时间，其实不难--- N graphs最少需要N-1 edges merge。 每次randomly选取两个graph链接，每个graph上随机选一个node。
需要自己定义graph数据结构。
写了个非最优解，每次merge以后copy了新的graph array。其实随机把一个graph merge进另一个，然后把需要delete的graph跟最后一个array element swap然后删除就可以。讨论的时候口头说了下但是没有时间改了。
