A CIDR block is represented as
IP/Prefix
复制代码
. The prefix tells you how many of the leading bits are fixed (the network portion), while the remaining bits are flexible (the host portion).
IPv4:
32 bits total.
Prefix /24:
The first 24 bits are fixed. If an incoming IP matches those 24 bits, it's in the range.
rules = [
("192.168.1.0/24" : "ALLOW"),
("8.8.8.8" : "DENY"),
#etc.
]
Write a function get_status(rules, "192.168.1.5") which returns the status of the given IP
in the above case - would pass since first 24 bits (192.168.1) matches and we hit the first rule -> ALLOW
This involved converting the IP address to binary string - checking for a match with the first /Prefix characters in the binary string, and then if it matches, returning the status.
rules list is ordered , so first rule hit is the one that you should return. Do a Quick refresh on Binary numbers.
