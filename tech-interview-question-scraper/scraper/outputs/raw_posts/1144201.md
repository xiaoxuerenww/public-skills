机票问题我记得很简单，你one pass就行了，其实类似buy and sell stock 1
就是一直记录一个min departure和一个min cost就行，因为可以同一天，每天就先把当前departure cost和min departure cost对比，然后和return cost相加，相加之后的值和前面的min cost比较就行，代码大概是这样
public int minCost(int[] departure, int[] return) {
int n = departure.length;
int minCost = Integer.MAX_VALUE;
int minDeparture = departure[0];
for (int i = 1; i < n; i++) {
minDeparture = Math.min(minDeparture, departure[i]);
minCost = Math.min(minCost, return[i] + minDeparture);
}
return minCost;
}
