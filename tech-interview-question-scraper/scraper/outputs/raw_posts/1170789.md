## Post

刚面完来报个dp，求加米 🙏
电面一共一轮，coding。
一上来面试官先问了句：
“Are you sure you want to do this in Java?”
然后直接贴题，基本就是 IP iteration / IP继承器 那道题。
跟 hack2hire 上那题大体一样，不过有一点区别：
Part 1 基本一致
Part 2 / Part 3 有变化
变化点是：step / increment 变成 constructor 里的一个参数传进去
这一点跟 hack2hire 上的不太一样，等于一开始设计类的时候就得考虑进去
我这边最后 做完了 Part 3。
做到后面问了下，面试官说：
一共 5 parts
所以我应该是只做到前 3 问，后面两问没来得及进 / 没做到。
最后结果是：
挂了
后面收到了拒信
个人感觉：
题本身如果提前刷过 hack2hire 确实有帮助
但如果默认按 hack2hire 的接口设计，看到 increment 放 constructor 这个变种会稍微卡一下
建议一开始先问清楚 总共有几问，不然时间分配会有点被动
用 Java 做也不是不行，但面试官一上来就问你是不是确定用 Java，感觉他们可能默认更多人用 Python 😂
求加米，后面如果大家感兴趣我可以再补一点我记得的细节。
